# Financial Performance Analysis (Budget vs Actual)

## Project Overview

This project analyzes financial data to compare **Budget vs Actual spending across different departments**.
The goal is to identify financial variance, monitor department performance, and generate insights for financial decision-making.

The project demonstrates a complete **data analytics workflow**, including data cleaning, analysis using Python, SQL-based queries, and dashboard visualization.

## Data Description

The dataset contains financial information with the following columns:

| Column     | Description              |
| ---------- | ------------------------ |
| Month      | Reporting month          |
| Department | Department name          |
| Budget     | Planned financial budget |
| Actual     | Actual spending          |
